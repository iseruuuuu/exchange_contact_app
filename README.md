# QR連絡交換アプリ　(タイトルは未定）. 

いろいろな連絡先を簡単に交換できるアプリとなっています。    
事前にIDやURLを登録しておけば、それに合わせたQRコードが生成できます。  
連絡先を交換するときに、相手はカメラを開くだけで連絡を交換することができます。
